This document provides AI prompts you can use to generate content for the TIU website pages.

## **Universal TIU Content Generation Prompt**

Use this prompt template for ANY page on the TIU website:

```
I need you to generate content for the [PAGE NAME] page of Tashkent International University (TIU) website.

**Context:**
- University: Tashkent International University (TIU)
- Location: Tashkent, Uzbekistan
- Target Audience: [prospective students / international students / current students / faculty / alumni]
- Page Purpose: [brief description of page purpose]

**Brand Guidelines:**
- Tone: Professional, welcoming, inspiring
- Voice: Authoritative but approachable
- Style: Clear, concise, informative
- Colors: Blue (#004080), White, Gold accent
- Focus: Academic excellence, international opportunities, career success

**Content Requirements:**
1. Main heading and subheading
2. Introduction paragraph (150-200 words)
3. [Number] key sections with headings and content
4. Statistics or highlights (if applicable)
5. Call-to-action (CTA) text
6. FAQs (if applicable)

**Page-Specific Information:**
[Add specific details about what this page should cover]

Please format the content in clean HTML using:
- <h2>, <h3> for headings
- <p> for paragraphs
- <ul><li> for lists
- <div class="info-box"> for highlighted content
- <a href="#" class="cta-primary"> for action buttons
```

## **Content Writing Guidelines**

When using these prompts, ensure the AI generates content that:

### **Tone & Style:**
- Professional yet approachable
- Inspirational and motivating
- Clear and concise
- Free of jargon (or explains technical terms)
- Inclusive and welcoming

### **Structure:**
- Clear headings and subheadings
- Short paragraphs (3-4 sentences)
- Bullet points for lists
- Visual breaks for readability
- Logical flow from general to specific

### **SEO Considerations:**
- Include relevant keywords naturally
- Use descriptive headings
- Include location-based terms
- Add meta descriptions
- Use semantic HTML

### **Call-to-Action:**
Every page should have clear CTAs:
- "Apply Now"
- "Request Information"
- "Contact Us"
- "Learn More"
- "Download Brochure"

### **Accessibility:**
- Plain language
- Descriptive link text
- Image alt text descriptions
- Proper heading hierarchy
- Sufficient color contrast

## **Tips for Best Results**

1. **Be Specific:** The more details you provide, the better the output

4. **Maintain Consistency:** Use similar prompts for similar pages
5. **Customize:** Adjust outputs to match TIU's actual offerings
6. **Verify Facts:** Always verify statistics and claims
7. **Local Context:** Add Uzbekistan-specific context where relevant
